# SiteSkin Privacy Policy

Last updated: June 16, 2026

SiteSkin applies visual refreshes to supported websites. It is designed to keep
the original website workflows intact while improving layout, spacing, and
readability.

## Data SiteSkin Uses

SiteSkin reads the current tab host so the popup can show whether a skin is
available for the site you are viewing. For supported sites, SiteSkin runs the
matching skin on that site.

SiteSkin stores extension status, skin pack metadata, and cached skin pack
assets locally in Chrome storage. This cache helps skins load reliably and
reduces repeated network requests.

## Remote Skin Packs

SiteSkin can fetch skin pack metadata, CSS, and JavaScript from the configured
GitHub repository. Chrome requires users to enable the per-extension "Allow user
scripts" setting before remotely updated skin scripts can run. Remote skin code
is limited to supported hosts listed in the extension package.

## What SiteSkin Does Not Collect

SiteSkin does not ask for account credentials, does not collect personal
information, does not collect browsing history, does not run analytics, and does
not sell or transfer user data.

SiteSkin does not send website form submissions or account data to the SiteSkin
author. Any normal website requests you make still go to the website you are
using.

## Third-Party Services

SiteSkin uses GitHub to retrieve public skin pack files. GitHub may receive
standard network request information, such as your IP address, when Chrome
requests those public files.
